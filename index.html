Development of a Cyber Physical System in Real Time Bolt Defect Detection

Machine Learning Based Automated Quality Control

Production Ready

🎯 Overview

An AI-powered cyber-physical system for real-time bolt quality inspection using YOLO object detection, Flask web interface, and Arduino hardware integration for automated quality control in manufacturing environments.

✨ Features

Cyber-Physical Integration

  • Real-time Detection with live camera streaming
  • Arduino-based physical alerts
  • Automated quality control with zero human intervention
  • Closed-loop system with instant response

Machine Learning & AI

  • YOLO v8 object detection
  • AI-based classification (Good/Defective)
  • Confidence scoring
  • Adaptive detection under varying conditions

Real-Time Processing

  • Live streaming with instant detection
  • Sub-second response time
  • Fixed detection zone
  • Duplicate prevention logic

Web-Based Monitoring

  • Flask dashboard
  • Real-time statistics
  • Color-coded visual feedback
  • Remote access

🏭 System Architecture

CYBER LAYER
Camera → YOLO Model → Flask Interface → Decision Logic

PHYSICAL LAYER
Arduino → Actuators → Physical Response (LED, Buzzer, Sorter)

📋 Requirements

  • Python 3.8+
  • OpenCV
  • PyTorch
  • Flask
  • Ultralytics YOLO
  • Arduino (optional)

🚀 Installation

Standard Setup

git clone <repository-url>
cd boltFaultDet
pip install -r requirements.txt

Run the System

python app.py

Access: http://localhost:5000

📊 Detection Logic

  • Good Bolt → Green
  • Defective Bolt → Red + Arduino alert
  • Single count per bolt
  • Real-time continuous inspection
