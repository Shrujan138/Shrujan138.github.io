🚀 Development of a Cyber-Physical System for Real-Time Bolt Defect Detection

🎓 Major Project | Electronics & Communication Engineering
📍 RV Institute of Technology and Management, Bengaluru
📅 Academic Year: 2025–2026

📌 Overview

This project presents an AI-powered Cyber-Physical System (CPS) designed for real-time bolt defect detection and automated quality control in industrial environments.

The system integrates:

🤖 Machine Learning (YOLOv8)
🎥 Computer Vision
⚙️ Embedded Systems (Raspberry Pi)
🔄 Automated Sorting Mechanism

It enables high-speed, accurate, and cost-effective inspection of bolts on a conveyor system without human intervention.

🎯 Problem Statement

Traditional bolt inspection methods:

❌ Manual and error-prone
❌ Time-consuming
❌ Inefficient for high-speed production

Advanced systems:

❌ Expensive (GPU/cloud dependent)
❌ Not suitable for edge deployment

👉 There is a need for a real-time, low-cost, and reliable inspection system.

💡 Proposed Solution

We developed a Cyber-Physical System that:

Detects bolt defects using YOLOv8
Runs entirely on edge device (Raspberry Pi)
Performs real-time sorting using actuators
Provides live monitoring through a web dashboard
🧠 Key Features
⚡ Real-time defect detection (~91.8% accuracy)
📦 Automated sorting of defective bolts
📊 Live dashboard monitoring
🔁 Temporal frame analysis for higher reliability
🔌 GPIO-based hardware integration
☁️ No cloud dependency (edge processing)

🔄 Working Principle
📸 Camera captures bolt images from conveyor
🧠 YOLOv8 model detects defects
🔍 Temporal filtering reduces false detection
⚙️ GPIO triggers actuator
📦 Defective bolts are separated
📊 Results displayed on dashboard
🧪 Dataset & Model
📁 Custom dataset collected from real environments
🏷️ Annotated for:
Cracks
Rust
Deformation
Missing bolts

🔧 Optimized for edge deployment
