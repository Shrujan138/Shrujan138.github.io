# 🚀 Development of a Cyber-Physical System for Real-Time Bolt Defect Detection

> 🎓 Major Project | Electronics & Communication Engineering  
> 📍 RV Institute of Technology and Management, Bengaluru  
> 📅 Academic Year: 2025–2026  

---

## 📌 Overview

This project presents an **AI-powered Cyber-Physical System (CPS)** designed for **real-time bolt defect detection and automated quality control** in industrial environments.

The system integrates:
- 🤖 Machine Learning (YOLOv8)
- 🎥 Computer Vision
- ⚙️ Embedded Systems (Raspberry Pi)
- 🔄 Automated Sorting Mechanism

It enables **high-speed, accurate, and cost-effective inspection** of bolts without human intervention.

👉 **Project Repository:**  
https://github.com/Shrujan138/Development-of-CPS-in-Real-Time-Bolt-Defect-Detection-using-ML-based-Automated-Quality-Control

---

## 🎯 Problem Statement

Traditional bolt inspection methods:
- Manual and error-prone  
- Time-consuming  
- Inefficient for high-speed production  

Advanced systems:
- Expensive (GPU/cloud dependent)  
- Not suitable for edge deployment  

👉 Need for a **real-time, low-cost, reliable inspection system**.

---

## 💡 Proposed Solution

A **Cyber-Physical System** that:

- Detects bolt defects using YOLOv8  
- Runs on Raspberry Pi (Edge AI)  
- Performs automated sorting using actuators  
- Provides real-time monitoring via dashboard  

---

## 🧠 Key Features

- Real-time defect detection (~91.8% accuracy)  
- Automated sorting system  
- Live dashboard monitoring  
- Temporal frame analysis (reduces false detection)  
- GPIO-based hardware integration  
- No cloud dependency  

---

## 🏗️ System Architecture

### 🔹 Hardware
- Raspberry Pi 4 Model B  
- Industrial Camera  
- Conveyor System  
- Actuator / Relay Mechanism  
- Lighting Setup  

### 🔹 Software
- YOLOv8 Model  
- Python Processing Pipeline  
- Flask Web Dashboard  
- OpenCV  

---

## 🔄 Working Principle

1. Camera captures bolt images  
2. YOLOv8 detects defects  
3. Temporal filtering improves accuracy  
4. GPIO triggers actuator  
5. Defective bolts are separated  
6. Results displayed on dashboard
